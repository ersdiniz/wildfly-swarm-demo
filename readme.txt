Olá!

Para rodar esta aplicação, basta executar o arquivo run.bat.

Este arquivo irá compilar, testar e rodar a aplicação utilizando o wildfly swarm.

Após a mensagem "WildFly Swarm is Ready" ser exibida no console, basta acessar
o endereço http://localhost:8080 para simular os exercícios.



No arquivo README.md constam mais detalhes das tecnologias utilizadas.
